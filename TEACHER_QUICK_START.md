### Step 1: Access the Platform
Visit the LanguageKids URL provided by your school administrator.

### Step 2: Create Your Account
1. Click **"Teacher Sign Up"**
2. Enter your school email address

### Step 3: Complete Your Profile
- Add your name
- Select your school
- Choose your grade level(s)
- Set your preferred language

## ğŸ“ Setting Up Your Class

### Create a New Class
1. Navigate to **"My Classes"** in the dashboard
2. Click **"Create New Class"**
3. Fill in the details:
   - **Class Name**: e.g., "Grade 3A - Morning"
   - **Age Group**: Select appropriate range
   - **Subject**: Language Learning
   - **School Year**: Current year

4. Click **"Create Class"**

### Generate Student Access Code
- Each class gets a unique 6-digit code
- Students use this code to join your class
- You can regenerate the code if needed

## ğŸ‘¥ Adding Students

### Method 1: Student Self-Enrollment (Recommended)
1. Share the class access code with students
2. Students visit the platform
3. They click **"Join Class"**
4. Enter the access code
5. Create their student account
6. You approve their enrollment

### Method 2: Manual Addition
1. Go to your class dashboard
2. Click **"Add Student"**
3. Enter student information:
   - Name
   - Student ID (optional)
   - Email (optional)
4. Click **"Add"**

### Managing Student Roster
- View all enrolled students
- Approve pending requests
- Remove students if needed
- Export student list

## ğŸ“š Assigning Modules

### Browse Available Modules
1. Click **"Module Library"**
2. Filter by:
   - Age group
   - Category (Animals, Colors, etc.)
   - Difficulty level
   - Estimated time

### Assign to Your Class
1. Select a module
2. Click **"Assign to Class"**
3. Choose your class
4. Set options:
   - **Start Date**: When students can begin
   - **Due Date**: Completion deadline
   - **Required**: Make it mandatory
   - **Quiz**: Include assessment

5. Click **"Assign"**

### Create Learning Paths
- Assign multiple modules in sequence
- Set prerequisites
- Create themed units
- Schedule releases

### Class Dashboard Overview
Your dashboard shows:

- **Average Progress**: Class completion rate

- Quiz scores
- Badges earned

- Time spent learning
- Strengths and weaknesses

   - Individual Student
   - Module Performance
   - Quiz Analysis

3. Choose date range
4. Click **"Generate"**
5. Download as PDF or CSV

### Real-Time Monitoring
- See who's currently active
- Track daily engagement
- Monitor quiz attempts
- View recent achievements

## ğŸ’¡ Tips for Success

### Best Practices

#### 1. Start Small
- Begin with 1-2 modules
- Let students explore the platform
- Gather feedback
- Gradually increase assignments

#### 2. Set Clear Expectations
- Explain the point system
- Show how to earn badges
- Demonstrate quiz features
- Set realistic deadlines

#### 3. Encourage Streaks
- Remind students to practice daily
- Celebrate streak milestones
- Create streak challenges
- Reward consistency

#### 4. Use Data Wisely
- Check dashboard weekly
- Identify struggling students early
- Celebrate top performers
- Adjust assignments based on data

#### 5. Engage Parents
- Share progress reports
- Explain the platform benefits
- Provide access codes
- Encourage home practice

### Common Challenges & Solutions

#### Challenge: Low Engagement
**Solutions:**
- Make it a class competition
- Set up leaderboards
- Offer real-world rewards
- Integrate with lesson plans

#### Challenge: Students Forget Passwords
**Solutions:**
- Use password reset feature
- Keep a class roster
- Teach password management
- Consider single sign-on

#### Challenge: Varying Skill Levels
**Solutions:**
- Assign different modules
- Use difficulty filters
- Create ability groups
- Provide extra support

#### Challenge: Technical Issues
**Solutions:**
- Check system requirements
- Clear browser cache
- Try different browser
- Contact support
